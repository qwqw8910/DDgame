# 甜甜的小秘密

多人互動工具平台，收錄多款小遊戲與生活工具，和朋友一起玩。

## 專案結構

```
vue-app/   ← Vue 3 前端
server/    ← Node.js 後端
docs/      ← 設計文件與規範
```

## 快速啟動

```bash
# 後端
cd server && node server.js

# 前端
cd vue-app && npm run dev

# 一鍵同時啟動前後端（根目錄）
npm run dev
```

## 近期更新
